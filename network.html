Ch 4: Networking and the Internet

Network Classifications

  • LAN: local area network
      • consists of a collection of computers in a single building or building complex
  • MAN: metropolitan area network
      • network of intermediate size (local community)
  • WAN: wide area network
      • links machines over greater distances (cities)

Network models

  • bus network: machines are all connected to a common communication line, the bus
  • star network: one machine serves as a central focal point to which all the others are connected

Protocols: rules by which activities are conducted

  • CSMA/CD Ethernet: Carrier Sense, Multiple Access with Collision Detection
      • dictactes that each message be broadcast to all the machines on the bus
  • CSMA/CA Wifi: Carrier Sense, Multiple Access with Collision Avoidance
      • wireless networks try to avoid collisions vs trying to detect them

Combining networks:

  • repeater: a device that passes signals back and forth between the two original buses (usually with some form of amplificatino) without considering the meaning of the signals
  • bridge: connects two buses and fowards a message across the connection only when that message is destined for a computer on the other side
  • switch: connects several buses; produces a network of several buses extending from the switch
  • router: special purpose computers used for forwarding messages
  • gateway: "point" at which one network is linked to an internet

Client/server model: popular convention used for interprocess communication

  • P2P: peer-to-peer: interprocess communication that provides service to and receives service from each other (instant messaging)
  • distributed system: software units that execute as processes on different computers
  • cloud computing: huge pools of shared computers on the network can be allocated for use by clients as needed

Internet Addressing

  • Tier 1: consist of high-speed, high-capacity, international WANs
  • Tier 2: more regional in scope and less potent
      • Tier 1 and 2 are essentially networks of routers that collectively provide the Internet's communication infrastructure.
  • Tier 3: an intermediary that provides access to the core

IP address: internet-wide addressing system that assigns a unique identifying address to each computer in the system

  • ICANN: Internet Corporation for Assigned Names and Numbers
  • DNS: domain name system
      • name servers are used as an Internet-wide directory system
  • TLD: Top-level domain: the suffix reflects the domain's classification
  • Country-code TDLS: country specific top-level domains

Internet protocol (IP): collection of protocol standards used by the Internet to implement the four-level communication hierarchy

  • FTP: File transfer protocol
      • an application for listing and copying files across the network
  • HTTP: Hyper text transfer protocol
      • application for transferring files over the web
  • SMTP: Simple mail transfer protocol
      • defines a way that two computers on the network may interact when transmitting an email message from one host to the other

World Wide Web: the combiniation of internet tech with hypertext

  • URL: Uniform resource locator
      • a unique address given to each document in order to retrive it on the Web
  • Client-side activities: activities performed by a browser
  • Server-side activities: activities performed by a webserver

Internet four-layer software

  • application layer: prepares message, provides destination address, and receives message
  • transport layer: chops message into packets, collects packets, and reassembles message
  • network layer: assigns intermediate address to each packet and detects that packet has reached its final destination
  • link layer: transfers and receives packet

Internet is:

  • packet-switching network: trasmission of data packets between devices over a shared network
  • fault tolerant: ability of a system to continue to operate without interruption when 1 or more components fail
  • redundant: most or all of the network remains connected even when very important tier1 routers fail

Cybersecurity: study of protecting computers,networks, and data from cycbercrime

  • malware: attacks through use of malicious software (viruses, worms, Trojan horses, and spyware)
  • virus: software that infects a coputer by inserting itself into programs that already reside in the machine
  • worm: autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
  • Trojan horse: program that enters a computer system disguised as a desirable program (such as a game or useful utility package)
  • spyware: collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
  • phishing: technique of obtaining infor explicity by simply asking for it
  • DDoS attack: distributed denial of service; attacks distributed over many hosts

Protection

  • firewall: a prevention technique to filter traffic passing through a point in the network
  • spam filter: firewalls designed to block unwanted email
  • proxy server: software unit that acts as an intermediary between a client and serve with the goal of shielding the client from adverse actions of the server
  • antivirus software: used to detect and remove the presence of known viruses and other infections
  • cryptography: study of sending and receiving messages securely in the presence of adversaries
  • public key encryption: techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure
